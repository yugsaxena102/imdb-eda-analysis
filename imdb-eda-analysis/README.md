# 🎬 Movies Dataset EDA | 30 Business & Analytics Questions

**End-to-end Exploratory Data Analysis on a 20,000+ movies dataset — answering 30 real-world business questions using Python, Pandas, and Seaborn.**

---

## 📌 Project Overview

This project performs structured EDA on a large, real-world movies dataset. The focus is not just visualization — it's **decision-focused analysis** that mirrors what data analysts do on the job: ask the right questions, clean messy data, and communicate findings clearly.

Built as a portfolio project to demonstrate readiness for **Data Analyst / Data Scientist** roles.

---

## 📊 Dataset Overview

| Attribute | Details |
|-----------|---------|
| Size | 20,000+ movies |
| Domain | Entertainment / Media Analytics |
| Source Format | CSV |

**Key Features:**
- Movie title & release year
- Genre & certificate (UA, PG-13, R, etc.)
- Duration (runtime in minutes)
- IMDb rating & vote count
- Director and lead stars
- Gross revenue

---

## 🎯 Business Questions Answered

The notebook answers **30 structured questions** across these analytical themes:

| Theme | Sample Questions |
|-------|-----------------|
| Genre Analysis | What genres dominate the industry? How has genre popularity shifted over decades? |
| Rating Trends | How have IMDb ratings evolved over time? What separates top-rated films from average ones? |
| Audience Engagement | Do high-vote-count movies get better ratings? What's the vote distribution? |
| Director Insights | Which directors consistently deliver high-rated films? |
| Runtime Analysis | What runtime range correlates with the best audience satisfaction? |
| Certificate Analysis | Are certain certificates (UA, PG-13, R) associated with higher ratings or revenue? |

Each question follows a structured format:
- 📌 Problem framing
- 🧠 Analysis logic
- 📈 Visualization
- 📝 Business interpretation

---

## 📈 Key Visualizations

- Rating and vote count distribution plots
- Genre-wise comparison charts
- Time-series trends across decades
- Director performance breakdowns
- Certificate vs. rating boxplots
- Correlation heatmaps (rating, votes, duration, revenue)

All plots are labeled, readable, and include written insights — not decoration.
