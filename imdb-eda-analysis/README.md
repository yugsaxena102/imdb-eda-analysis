# IMDb Movie EDA 🎬📊

Exploratory Data Analysis on IMDb movie dataset using Python and Pandas.

## What This Project Covers

- Data cleaning (duration, votes, gross income conversion)
- Handling missing values
- Finding high-rated & low-voted movies
- Overhyped movies (low rating, high gross)
- Genre-wise analysis (ratings, gross income)
- Certificate-wise analysis
- Year-wise movie count trends
- Correlation: duration vs rating, votes vs gross income
- Highest grossing directors

## Tech Stack

- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

## Dataset

IMDb movies dataset with columns:
`id, title, year, certificate, duration, genre, rating, votes, gross_income, director_id`

## How to Run

```bash
pip install pandas matplotlib
jupyter notebook project_imdb.ipynb
```
